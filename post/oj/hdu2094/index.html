HDU2094 - 产生冠军

2019-12-08 约 563 字 预计阅读 2 分钟

链接：https://vjudge.net/problem/HDU-2094

Task

有一群人，打乒乓球比赛，两两捉对撕杀，每两个人之间最多打一场比赛。 球赛的规则如下： 如果A打败了B，B又打败了C，而A与C之间没有进行过比赛，那么就认定，A一定能打败C。 如果A打败了B，B又打败了C，而且，C又打败了A，那么A、B、C三者都不可能成为冠军。 根据这个规则，无需循环较量，或许就能确定冠军。你的任务就是面对一群比赛选手，在经过了若干场撕杀之后，确定是否已经实际上产生了冠军。

Input

输入含有一些选手群，每群选手都以一个整数n(n<1000)开头，后跟n对选手的比赛结果，比赛结果以一对选手名字（中间隔一空格）表示，前者战胜后者。如果n为0，则表示输入结束。

Output

对于每个选手群，若你判断出产生了冠军，则在一行中输出“Yes”，否则在一行中输出“No”。

Solution

这种计算冠军的方法其实是：有且仅有一个人从未输过，他就是冠军。 所以用两个无序集合，一个all记录所有参赛者，一个lose记录输过的人。如果|all|-|lose|==1，那么多出来的那一个就是冠军。

#include<cstdio>
#include<string>
#include<iostream>
#include<unordered_set>
using namespace std;
int main(){
    int N;
    while(scanf("%d",&N)!=EOF){
        if(N==0) return 0;
        else{
            unordered_set<string> all,lose;
            for(int i=0; i<N; ++i){
                string winner,loser;
                cin>>winner>>loser;
                all.insert(winner);
                all.insert(loser);
                lose.insert(loser);
            }
            if(all.size()-lose.size()==1) printf("Yes\n");
            else                          printf("No\n");
        }
    }
    return 0;
}
